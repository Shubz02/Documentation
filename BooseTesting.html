AppCanvasBase

Provides an abstract base class for a canvas with drawing methods. Includes functionality for setting color, moving the cursor, and drawing shapes and text.

Form1

The Form1 class is responsible for providing a UI interface for Duncan's BOOSE Interpreter. It allows the user to enter drawing commands, interpret those commands, and render the output on a canvas.
