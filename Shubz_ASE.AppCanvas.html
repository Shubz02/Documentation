Class AppCanvas

Namespace
Shubz_ASE

Shubz ASE.dll

Provides a customizable canvas control for drawing, managing, and saving shapes, images, and text in a Windows Forms application. The AppCanvas class includes functionality for setting pen properties, drawing various shapes, and hosting an HTTP API server to allow remote control of the canvas.
