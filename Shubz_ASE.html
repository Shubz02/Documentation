AppCanvas

Provides a customizable canvas control for drawing, managing, and saving shapes, images, and text in a Windows Forms application. The AppCanvas class includes functionality for setting pen properties, drawing various shapes, and hosting an HTTP API server to allow remote control of the canvas.

Form1

The main form for the BOOSE Drawing Application, providing an interactive interface for drawing shapes, setting colors, adjusting brush thickness, and evaluating mathematical expressions. The Form1 class enables users to work with an AppCanvas for drawing, and includes text-based input fields for color and mathematical expression evaluation.
